# Trading Terminal (Python)

This is a simple Python CLI project created for learning modular programming.

## Features

1. Risk Calculator
2. Position Size Calculator
3. Profit / Loss Calculator
4. Compound Growth Calculator

## Project Structure

a_main.py → main menu  
b_risk_menu.py → risk calculation  
c_position_size_menu.py → position size calculation  
d_profit_loss_menu.py → profit / loss calculation  
e_compound_menu.py → compound growth calculation  

## Run the project


```bash
python a_main.py
```
